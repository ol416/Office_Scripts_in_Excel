# Office Script in Excel 项目

## 项目概述

本项目是一个个人项目，旨在探索和使用Office Script在Excel中的功能。

## 功能

- 使用Office Script自动化Excel中的任务。
- 提高工作效率。
- 个人项目，用于学习和实践。

## 使用方法

1. 在Excel中打开需要自动化的工作簿。
2. 打开自动执行，并创建一个新脚本。
3. 粘贴脚本到代码编辑器中。
4. 运行脚本并查看结果。

## 注意事项
- 项目中大部分为AI生成，可能存在错误。
- 请确保在使用Office Script之前已经熟悉Excel的基本操作。
- 本项目为个人项目，不应用于商业用途。
